# Constants 

### Objective

In this project, our main goal is to understand constants in Java.

### Problem
Develop a program that converts temperature between Celsius and Fahrenheit using constants for:
- The ratio of the Fahrenheit degree size to the Celsius degree size (1.8)
- The freezing point of water in Fahrenheit (32)


### Implementation
- Create **constants** for the **conversion ratio and freezing point**.
- Create a `double` variable `celsiusTemp` and assign it the value `25.0`.
- Create a `double` variable `fahrenheitTemp` and assign it the value `77.0`.
- Convert Celsius to Fahrenheit using `°F = (°C × conversion ratio) + freezing point` and print the result.
- Convert Fahrenheit to Celsius using `°C = (°F - freezing point) /conversion ratio` and print the result.
